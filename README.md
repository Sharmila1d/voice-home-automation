# Voice Controlled Smart Home Automation

## Overview

A Python-based Voice Controlled Smart Home Automation project developed using Speech Recognition and Webots.

## Features

- Voice Recognition
- Command Parsing
- Smart Home Automation
- Webots Simulation
- Light Control
- Fan Control
- AC Control

## Technologies Used

- Python
- Webots
- SpeechRecognition
- PyAudio

## Commands

- Turn on light
- Turn off light
- Turn on fan
- Turn off fan
- Turn on AC
- Turn off AC

## Future Improvements

- IoT Integration
- MQTT
- ROS Integration
- Mobile App
